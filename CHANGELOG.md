# Changelog
All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-01
### Added
- `verition-worktree-manager.openSubpath`: open worktrees in a specific subfolder (e.g. `analytics`)
- Open **internal**/**external** terminal in subfolder
- Reveal subfolder in system explorer
- Optional post-create step to write subfolder `.vscode/settings.json`
### Changed
- Renamed command/IDs to avoid collisions with upstream
### Notes
- Forked from `jackiotyu/git-worktree-manager` at upstream tag `v3.15.0`. See “Upstream history” below.
